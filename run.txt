First-time setup

Backend install
python3 -m venv .venv
source .venv/bin/activate
pip install -r backend/requirements.txt
cd backend
uvicorn main:app --reload --port 8000


Frontend install
cd frontend
npm install
npm run dev


========================================
After installation, run only these
========================================

Terminal 1 - backend
source .venv/bin/activate
cd backend
uvicorn main:app --reload --port 8000

Terminal 2 - frontend
cd frontend
npm run dev
